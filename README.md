# Dashboard Application

A modern, visually impressive dashboard built with React, Vite, and Recharts.

![Dashboard Screenshot](screenshot.png)

## Features

- **Data Visualization**: Beautiful charts and graphs using Recharts
- **Responsive Design**: Works on all device sizes
- **Modern UI**: Clean, card-based layout with subtle animations
- **Performance**: Fast loading with Vite development tooling
- **Interactive Components**: Hover effects and smooth transitions

## Tech Stack

- React 19
- Vite
- Recharts for data visualization
- CSS3 for styling

## Key Components

1. **Stats Cards**: Display key metrics with trend indicators
2. **Sales Overview Chart**: Bar chart showing sales performance
3. **User Devices Chart**: Pie chart showing device distribution
4. **Performance Metrics Chart**: Line chart tracking key performance indicators
5. **Recent Activity Feed**: Timeline of recent events

## Getting Started

1. Install dependencies:

   ```
   npm install
   ```

2. Start the development server:

   ```
   npm run dev
   ```

3. Open your browser to the provided URL (typically http://localhost:5173)

## Building for Production

```
npm run build
```

This will create a production-ready build in the `dist` folder.

## Previewing Production Build

```
npm run preview
```
